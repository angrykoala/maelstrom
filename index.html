Maelström

Economy and trading MMO-RTS(Massively Multiplayer Online - Real Time Strategy) simulation videogame under Node.js.

This Open-Source project aims to provide a basic framework and solutions for MMO videogames providing multiple independent services

Services

Maelström is composed of several which provides the features with low interdependence between them, this is a list of both, implemented and planned microservices and their current status. Each microsevice is in a different repository

Service Status Description
Users Finished Users login/signup and sessions
World Testing World server and game logic
Web In Progress Game web client
Messages Not Started Game messaging system

Each microservice can be deployed independently, the instructions of each microservice is in its own repository.
